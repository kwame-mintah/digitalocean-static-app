About

DigitalOcean offers an App Platform, as part of the starter they can host up to 3 static websites for free. This project has been created to make use of this service and is just an example. The aim is to just create a demo site and deploy it successfully via Terraform.

Ideally, each environment will have a different static website deployed/updated. However at this moment in time the same static site will be deployed to each environment. Additionally any new code committed to this project 'master' branch will be automatically deployed.

Please note that this site and files is just for demonstration purposes, all images and fonts used belong to their respective owners.
